# 微信小程序-知否精选

知乎日报小程序客户端

## 功能

- 查询文章列表

- 阅读文章

- 查看文章评论

- 分享

## 预览

知否精选已发布线上版本，扫描二维码直接打开小程序
![logo](images/qr_code.jpg)

## 使用

克隆本项目 -> 在微信开发工具中添加项目 -> 选择项目目录
